### Docker 快速部署 Redis Cluster 集群

#### 服务器

```
server1 192.168.1.1
server2 192.168.1.2
```

#### 快速启动容器

两台服务器分别执行

```
$ src/quickstart.sh
```

#### 查看容器

```
$ docker ps
```

#### 创建集群

登录任意一台服务器执行

```
$ src/redis-cli --cluster  create 192.168.1.1:7000 192.168.1.1:7001 192.168.1.1:7002 192.168.1.2:7000 192.168.1.2:7001 192.168.1.2:7002  --cluster-replicas 1
```
